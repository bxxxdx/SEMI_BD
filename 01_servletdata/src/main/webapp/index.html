서버에서 데이터를 저장하는 객체

  • HttpServletRequest : 1회용 데이터를 저장하는 저장소
    * 1회용 => 요청하고 응답이 완료될때 까지만 사용
  • HttpSession : 원하는 시점까지 데이터를 유지하는 저장소
    * 개발자가 원하는 시점에 삭제하거나 혹은 설정값에 의해 자동으로 삭제될 때 까지 이용
    * 로그인에 주로 쓰인다, 혹은 드물지만 오늘 본 상품 등에도 이용할 수는 있다.
  • ServletContext : 서버가 실행되고 종료될때까지 유지하는 저장소
  • 우선순위는 적힌 순서대로라고 보면 된다. 클라이언트 쪽에 저장되는 요소는 쿠키도 있음

저장 방법
setAttribute(String, Object) 메소드로 key:value 형식으로 저장함
getAttribute("key") 메소드로 값을 가져옴 -> * 반환형이 Object로 형변환을 해서 사용해야 함.
